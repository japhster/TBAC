Items

An Item is a named, interactable object within your world. They can be picked up inspected, given to or received from friends or dropped by enemies. They can also be defined as in a room but not takeable.

Descriptions

There are two ways of describing an item.

The first way is the actual description of the item as the player will see if they inspect it in their inventory or in the location it's found.

The second is how the item should be described when the player is presented with the description of the room. This is know as the "In Room Description" and is added onto the end of the room's description.

Item Type

When creating an item, you can select the type of the item. This will define what can be done with the item in the world.

A Generic item is just any old item, from a cabbage to a letter to a pile of gold. These items have no definite purpose but may still be dropped by enemies or given to/received from friends.

An item defined as a Key can be used in a "Use" command to unlock a connection between two locations. This could be a keycard, or a literal key or a stone tablet placed into an alter.

A Container is an item that can store one or more other items and can be used in an "Open" command to retrieve the items inside. For example a box, or a treasure chest.

A Weapon is an item that must have a defined damage it can deal as these items can be used in combat against enemies. "The Sword of Destiny" is a weapon that deals great blows to your foes.

Item Location

There are five ways of defining how a player can come across an item in your world.

  • One is that they can simply start with the item in their inventory as soon as they begin the adventure.
    • This can be defined by checking the "Is Starting item" box on item creation
  • Another is it can be found in a room they enter.
    • This can be defined by adding the room during creation
  • The player could also open an item that contains the item.
    • This can be achieved by adding the container as the "Contained within" field in the item creation
  • Through a conversation with a Friend the player could receive an item.
    • This can be done during the setup of the Friends dialogue options.
  • Finally, the player could defeat an enemy and on death they drop an item.
    • This is setup when creating an enemy, at which point the item can be added to the enemy's drops.

The only times where the "In Room Description" is relevant is for if the item is found in a room, or an enemy drops the item. In the other cases, the base Description is the only one the player will see.

Example

You have an item you have named The sword of Destiny. Accepted names for the item include "sword" and "weapon". You want to delay the discovery by the player that this is "The Sword of Destiny" so you add an in room description of "An inconspicuous sword lies on the ground" but a base description of "It's the sword of Destiny! Many tales are told of it's brilliance".
