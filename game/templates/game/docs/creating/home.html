Welcome to a guide on how to create your very own text based adventure game. This documentation will take you through the various components that are required to build your world.

Before you begin though, a note on names: most of the components in your world will be named, but in order to provide a smooth experience for your players you may wish to provide a list of alternative names to refer to them, this is possible using the "Accepted Names" fields which should be a comma-separated list of references. For example you may have an item you want to call "The Sword of Destiny", this is probably a bit long to type out each time, so you might wish to add an accepted name "sword".
